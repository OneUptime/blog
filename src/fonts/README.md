# Fonts for Social Media Image Generation

This directory contains the Poppins Bold font used for generating social media images.

## Font

- **Poppins Bold** - Geometric sans-serif with a modern, clean aesthetic

## Setup

The `Poppins-Bold.ttf` font should already be present in this directory. If it's missing, download it from:

1. Google Fonts: https://fonts.google.com/specimen/Poppins
2. Or run this command:

```bash
curl -L -o src/fonts/Poppins-Bold.ttf "https://github.com/google/fonts/raw/main/ofl/poppins/Poppins-Bold.ttf"
```
